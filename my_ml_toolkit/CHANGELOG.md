# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-07

### Added
- Initial release of My ML Toolkit
- Enterprise-grade machine learning evaluation pipeline
- Support for classification and regression tasks
- Automatic model selection and hyperparameter tuning
- Two-phase evaluation (quick screening + full tuning)
- SMOTE support for imbalanced data handling
- Comprehensive visualizations (learning curves, confusion matrices, feature importance)
- Dynamic preprocessing routing per model
- Automatic data splitting (train/val/test) with sequential or random methods

### Code Quality
- Input validation with clear error messages
- DRY refactoring eliminating code duplication
- Helper functions for maintainability
- Defensive copying preventing data mutation
- Support for Gradient Boosting models (XGBoost, LightGBM, CatBoost)

### Documentation
- Comprehensive README with usage examples
- API reference documentation
- Troubleshooting guide
- Code quality & refactoring section
